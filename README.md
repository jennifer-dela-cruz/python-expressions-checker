# About
This program evaluates the arithmetic expression and prints the result 

# Running python-expressions-checker
python3 jh-delacruz-02.py

# Character Set
This program uses ASCII characters

# Operator Symbols
Addition: PLUS x y

Subtraction: MINUS x y

Multiplication: TIMES x y

Division: DIVBY x y

Modulo:  MODU x y

# Other Commands
Starts the program: CREATE

Ends the program: RUPTURE

Prints a string: GIVEYOU! string

Prints a string with new line: GIVEYOU!! string
